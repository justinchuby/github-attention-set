# Privacy Policy — GitHub Attention Set

**Last updated:** May 5, 2026

## Summary

GitHub Attention Set does not collect, store, or transmit any personal data to any third party. All data stays on your device.

## What data is stored

- **GitHub Personal Access Token** — stored locally in `chrome.storage.local` on your device only. Never synced to Google servers or any third party.
- **Cached PR data** — PR titles, statuses, and timeline events fetched from GitHub API are cached locally for instant popup loading.
- **User preferences** — debounce time, poll interval, notification settings, repo filters. All stored locally.

## What data is transmitted

- **To GitHub API only** — the extension communicates exclusively with `https://api.github.com` using your Personal Access Token to fetch pull request data. All communication is over HTTPS.
- **No other network requests** — no analytics, no tracking, no telemetry, no third-party services.

## What data is NOT collected

- No browsing history
- No personal information
- No usage analytics
- No crash reports
- No data sold or shared with any third party

## Data retention

All data is stored locally on your device and can be cleared at any time by removing the extension or clearing extension storage.
